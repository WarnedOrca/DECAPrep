🚀 DecaPrep AI

DecaPrep AI is an AI-powered learning platform designed to help DECA members prepare for competitions through personalized practice, Performance Indicator mastery, and realistic exam simulations.

🎯 Purpose

DecaPrep AI supports DECA’s mission of preparing emerging leaders and entrepreneurs in marketing, finance, hospitality, and management by providing:

Structured, high-quality study tools
Realistic competition-style practice
AI-driven personalized learning
✨ Features
📚 Performance Indicator (PI) System
Comprehensive database of DECA Performance Indicators
Filter by:
Career Cluster (Marketing, Finance, Hospitality, etc.)
Topic / Instructional Area
Targeted practice by weak areas
🧠 AI-Powered Study Tools
Flashcards (term-based and scenario-based)
Custom quizzes (user-defined length & difficulty)
AI-generated explanations for deeper understanding
🧪 Exam Simulation
Generate realistic DECA-style exams:
10 / 25 / 50 / 100 MCQs
Timed practice mode
Instant scoring + answer explanations
Performance breakdown by topic
🤖 AI Coach
Ask questions about business concepts
Get explanations for Performance Indicators
Practice roleplay scenarios (like DECA events)
Personalized study recommendations
📊 Dashboard & Analytics
Track strengths and weaknesses
Monitor progress over time
Identify areas for improvement
Adaptive learning recommendations

🚀 Getting Started
# Clone the repository
git clone https://github.com/your-username/deca-prep-ai.git

# Navigate into the project
cd deca-prep-ai

# Install dependencies
npm install

# Run development server
npm run dev
